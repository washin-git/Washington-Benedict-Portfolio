As software systems become increasingly complex, effective management of this complexity is vital. The study presents the Code Complexity Measuring Tool (CCMT), which addresses challenges in assessing code complexity by incorporating nine key factors: size, control structure type, nesting level, inheritance level, try-catch blocks, recursion, array declarations, compound conditionals, and input-output statements. By assigning distinct weightings to these elements, CCMT facilitates precise complexity estimation. Its adaptability makes it a forward-looking tool, suitable for integration into advanced fields like AI-driven software development, ensuring its relevance in the dynamic software engineering landscape.
